# Podigee Theme SDK

An easy way to develop Podigee Website Themes locally.

## Usage

### Install dependencies

```
yarn install
```

### Run local server

```
./sdk.js $apiKey $podcastId
```

- You can find the API key in your user settings. If you can't see it, please contact us.
